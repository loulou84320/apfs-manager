# 🍎 APFS Manager for Linux

Outil complet pour **lire et écrire** sur des disques Apple APFS sous Linux.
Supporte les volumes **chiffrés (FileVault)** avec saisie de mot de passe sécurisée.

---

## 📋 Prérequis

- Linux (Ubuntu/Debian/Fedora/Arch/openSUSE)
- Python 3.8+
- Connexion internet (pour l'installation)
- Droits sudo

---

## 🚀 Installation rapide

```bash
git clone https://github.com/loulou84320/apfs-manager.git
cd apfs-manager
chmod +x install.sh
./install.sh
